obstacle_detector with vehicle_description

A demo which shows the obstacle_detector with vehicle_description.

status = working

To Run

roslaunch system demoA.launch

Expected Output

Gazebo should pop up, and the vehicle is located at the origin and heading towards y axis. There should be three cylinder obstacles ahead of the vehicle. Rviz should display the pointcloud in blue and the fitted circular obstacles.
